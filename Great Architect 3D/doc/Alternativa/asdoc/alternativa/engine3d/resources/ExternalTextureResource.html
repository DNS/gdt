Packagealternativa.engine3d.resources
Classpublic class ExternalTextureResource
InheritanceExternalTextureResource TextureResource Resource Object

a TextureResource, that uses data from external source (file). Link for this source is specified on instance creation. Used with TexturesLoader, which gets ExternalTextureResource for uploading needed files. Size of texture must be power of two (e.g., 256Ñ…256, 12812, 25632).
